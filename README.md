# 2026 JACC Cardiovascular Statistics in the United States Report

The code in this repository performs the statistical analyses and produces the associated figures for each of the data briefs in the *2026 JACC Cardiovascular Statistics in the United States Report*. The methodology for these analyses is describe in more detail [here] (https://www.jacc.org/doi/10.1016/j.jacadv.2025.102548).

To execute the code, please navigate to the repository for a specific project, unzip the csv file, and run the entire R script. Please note that the healthcare access, affordability, and utilization manuscripts all use one common R script.
